# 🔥 AI Product Recommendation System

An AI-powered web application that generates product recommendations and price insights dynamically using OpenAI API.

---

## 🚀 Project Overview

This system allows users to:

- Enter a product category (AC, fridge, geyser, etc.)
- Get AI-generated product recommendations with specifications
- Select a product and view price trends (2023–2026)
- View estimated future pricing (AI-based prediction)

👉 The system does NOT use a database. All data is dynamically generated using AI.

---

## 🧠 Features

- 🔍 Product recommendation using AI
- 📊 Price trend analysis (2023–2026)
- 💰 Prices displayed in INR (₹)
- 🔮 Future price prediction (2026)
- 🔐 Password-protected access
- ⚡ Fast and interactive UI using Streamlit

---

## ⚙️ Tech Stack

- Python
- Streamlit
- OpenAI API

## ▶️ How to Run Locally

1. Install dependencies:

pip install streamlit openai


2. Add your OpenAI API key in the code

3. Run the app:

streamlit run app.py
